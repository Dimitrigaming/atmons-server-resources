===========================================
  ATMons Server - KubeJS Script Installer
===========================================

INSTRUCTIONS:
-------------
1. Download "custom_coins.bat"

2. Open CurseForge and find the ATMons modpack

3. Click the three dots (...) next to the modpack
   and select "Open Folder"

4. Place "custom_coins.bat" into that folder

5. Double click "custom_coins.bat" to run it

6. Fully close and restart Minecraft

7. You should now be able to join the server!


NOTE: If you get a new "Connection Lost" error
after a server update, just run the bat file
again to get the latest scripts.
